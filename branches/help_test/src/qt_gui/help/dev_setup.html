<> The first step in configuring linuxtrack is configuration of the tracking device.
Select the device of your choice from the "Tracking device" combobox.
If you can't find the device you want to use, try to press the "Refresh" button and check again.

Webcam Setup

If you chose webcam as your tracking device, start by setting the Pixel Format. The preferred format is YUYV, but you can experiment to see which one works for you best.

Then select the desired Resolution & Framerate.

To check, how the tracking works, go to the Tracking window and select Camera view pane. Press Start button to comence tracking.
Ideally, you should see three (or one, in case of single point model) "blobs" - these are light emitting or reflecting points of your model seen by the camera. Each of those "blobs" should contain a white cross to mark a valid blob.

However, to setup the webcam for the first time, you have to set few more things...
If you see more light sources, you can try one of these solutions.

  • Try to remove unnecessary sources of light in the webcam's field of view
  • You can set lower exposition value
  • You can set the Threshold value higher
  • You can use piece of film or magnetic tape as a light filter.

When you have removed as much interfering light sources as possible, it is a time to select a blob size constraints.
These limits are used to disctiminate false "returns" by applying a constraints on a number of pixels that form a blob.
If a blob should be valid, but it still doesn't contain the white cross, then try lowering the Min. blob value and increasing the Max. blob value. It is best to finetune the Min. blob value when being farthest away from the camera, while the Max. blob value while being closest to the camera.

Then last, but not least, there is a possibility to flip the images from webcams (mainly some net/notebooks have webcam mounted upside down).

Troubleshooting tips

  • Make sure the webcam is correctly focused
  • Use an application able to controll brightness, exposure ad so on
  • If the webcam is not shown in devices combo, make sure thad you have a permissiion to access it
